Privacy Policy

Gas Estimator · by Bitcoineo

Data Collection

Gas Estimator does not collect, store, or transmit any user data. There are no analytics, no tracking, and no telemetry of any kind.

Network Requests

The only network request the extension makes is to the Blocknative public Gas Platform API to fetch current Ethereum gas prices. No user information is included in this request.

No data leaves your browser except this single API call.

Local Storage

User preferences (such as the EIP-1559 toggle) are stored locally on your device using chrome.storage.local. This data never leaves your browser and is not accessible to any external service.

Summary

  • No personal data collected
  • No analytics or tracking
  • No third-party scripts
  • Preferences stored locally only
  • One external API call (Blocknative) for gas prices
